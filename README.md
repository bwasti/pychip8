# pychip8

A CHIP-8 emulator written in Python using pygame

## Try it out
```
python chip8/main.py --rom-path roms/Breakout\ \[Carmelo\ Cortez\,\ 1979\].ch8
```

![](https://media.giphy.com/media/QVyjipq9sdU9xPojBP/giphy.gif)

## Debugger and Stepper
```
--debug
--stepper
```
